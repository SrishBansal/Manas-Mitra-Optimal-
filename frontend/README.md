# MindCare - AI Mental Health Companion

A compassionate AI-powered mental health chatbot designed to provide immediate support, guidance, and professional referrals for individuals seeking mental health assistance.

## 🌟 Features

### 🤖 AI-Powered Conversations
- Empathetic AI companion trained for mental health support
- Context-aware responses based on user input
- Crisis detection and immediate professional referral
- Anonymous and secure conversations

### 🧠 Psychological Assessments
- Evidence-based screening tools for anxiety, depression, and stress
- Immediate results with personalized recommendations
- Progress tracking and insights
- Professional referral suggestions based on results

### 🧘 Relaxation Techniques
- Guided breathing exercises (4-7-8, Box Breathing, Calm Breathing)
- Mindfulness and meditation sessions
- Progressive relaxation techniques
- Quick stress relief tips and daily practices

### 👥 Professional Referral System
- Database of qualified mental health professionals
- Location-based search and filtering
- Specialization matching
- Crisis support resources and emergency contacts

### 🔒 Privacy & Security
- Anonymous user IDs (no personal data stored)
- Secure, encrypted conversations
- GDPR compliant data handling
- No tracking or data collection

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd mental-health-chatbot
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🧪 AI Response System

The chatbot uses a sophisticated intent detection system:

1. **Crisis Detection**: Identifies keywords indicating immediate danger
2. **Mental Health Assessment**: Recognizes symptoms and triggers
3. **Empathetic Responses**: Provides appropriate emotional support
4. **Professional Referral**: Connects users with qualified professionals

### Response Categories
- **Crisis**: Immediate professional intervention
- **Assessment**: Psychological screening recommendations
- **Support**: General mental health guidance
- **Information**: Educational resources and tips
